## Téléchargement

Dernière version → onglet **[Releases](../../releases/latest)**.

| OS | Fichier |
|----|---------|
| **Windows** | `Horus_x.y.z_x64-setup.exe` ou `_x64_en-US.msi` |
| **macOS** (Apple Silicon / Intel) | `Horus_x.y.z_aarch64.dmg` / `_x64.dmg` |
| **Linux** | `Horus_x.y.z_amd64.AppImage` ou `_amd64.deb` |

## Installation

- **Windows** : SmartScreen peut avertir (app non signée) → *Exécuter quand même*.
- **macOS** : clic droit → *Ouvrir* au 1er lancement.
- **Linux** : `.AppImage` (`chmod +x`) ou `.deb`.

## Mises à jour

L'app est une **coquille web** : les mises à jour fonctionnelles arrivent automatiquement via le déploiement web. Seul un changement de la coquille elle-même nécessite de réinstaller le binaire.
