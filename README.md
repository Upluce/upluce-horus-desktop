# Horus Desktop

Fenêtre desktop de **Horus**, la supervision d'infrastructure d'Upluce. Construite avec **Tauri 2** ; charge l'interface web Horus (`horus-app.ignira.fr`).

> Dépôt **public de distribution** : binaires uniquement. Code source dans `Upluce/upluce.com` (`apps/horus-desktop`).

## Téléchargement

Dernière version → onglet **[Releases](../../releases/latest)** : Windows (`.msi`/`.exe`), macOS (`.dmg` arm64 + x64), Linux (`.deb`/`.AppImage`).

> L'app étant une coquille web, les mises à jour fonctionnelles arrivent via le déploiement web ; seul un changement de la coquille nécessite une réinstallation.
